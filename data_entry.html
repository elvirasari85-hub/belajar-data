Modul 1: Data Entry & Pengelolaan Dasar

Dari Basic hingga Advance: Kuasai fondasi data yang bersih.

Kurikulum Modul

Level Basic: Pengenalan Data & Spreadsheet

Pelajaran 1.1: Apa Itu Data dan Pentingnya Kebersihan

Data adalah aset terpenting. Pelajaran ini membahas dasar-dasar data dan alasan mengapa data kotor (tidak akurat, duplikat) dapat merugikan bisnis.

  • Pengertian **Data Struktural vs. Non-Struktural**.
  • **Dampak Missing Value** pada analisis.
  • Prinsip Dasar **Validasi Input** saat memasukkan data.

Pelajaran 1.2: Fungsi Wajib di Excel / Google Sheets

Kuasai fungsi spreadsheet yang harus diketahui setiap profesional data. Fokus pada efisiensi dan akurasi perhitungan.

  • Menggunakan **SUM, AVERAGE, MAX, MIN** untuk statistik deskriptif.
  • Memanfaatkan **VLOOKUP/XLOOKUP** untuk menggabungkan data dari tabel berbeda.
  • Menganalisis data dengan cepat menggunakan **Pivot Table**.

Level Intermediate: Teknik Data Cleaning (Pembersihan Data)

Pelajaran 2.1: Menghapus Duplikasi dan Mengatasi Missing Value

Pelajari teknik praktis untuk membersihkan *dataset* yang besar dan tidak rapi.

  • Fitur **Remove Duplicates** (Hapus Duplikasi) di Excel/Sheets.
  • Teknik mengisi *Missing Value* dengan **rata-rata** atau nilai **modus**.
  • Menggunakan fungsi **TRIM** dan **CLEAN** untuk membersihkan spasi dan karakter tersembunyi.

Pelajaran 2.2: Normalisasi Format Data (Tanggal, Mata Uang, Teks)

Menyeragamkan format adalah kunci untuk analisis yang akurat. Data yang tidak konsisten tidak dapat diolah.

  • Mengubah format tanggal agar **konsisten** (misalnya dari 1-1-2024 menjadi 01/01/2024).
  • Memisahkan teks ke kolom berbeda menggunakan **Text to Columns**.
  • Mengubah huruf besar/kecil (**UPPER/LOWER/PROPER**).

Level Advance: Integrasi dan SQL Dasar

Pelajaran 3.1: Pengenalan SQL (Structured Query Language)

SQL adalah bahasa standar untuk database. Mempelajari dasar SQL memungkinkan Anda berinteraksi dengan data di luar spreadsheet.

  • Perintah dasar: **SELECT, FROM, WHERE** (Mengambil data).
  • Pengenalan konsep **Primary Key** dan **Foreign Key**.
  • Latihan dasar menggunakan platform SQL online gratis.

Pelajaran 3.2: Studi Kasus Lintas Platform

Pelajaran ini menghubungkan Spreadsheet (Excel/Sheets) dengan konsep Database (SQL) dan migrasi data.

Anda akan belajar bagaimana mempersiapkan file **CSV** (Comma Separated Values) agar siap diimpor ke sistem database apa pun.
